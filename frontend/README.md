
# Crypto Tracker - Frontend

Qui di seguito delle brevi indicazioni sul come avviare il progetto in locale.


## Elenco dei comandi da terminale

In ordine:

#### Installare le librerie

```http
  npm install
```

#### Buildare il progetto

```http
  npm run build
```

#### Avviare il progetto

```http
  npm run start
```
oppure

```http
  ng serve
```
